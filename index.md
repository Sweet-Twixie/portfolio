## [Persistent Homology on Breat Cancer histopathology image dataset](BreakHis-persistent-homology/index.html)

This project explores the use of **persistent homology** for **binary classification** of breast cancer histopathology images from the **BreakHis** dataset. The project demonstrates how topological data analysis (TDA) can capture global structural patterns in medical images and outperform classical handcrafted texture descriptors.
